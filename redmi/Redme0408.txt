ai_router 폴더 전체 흐름 분석
프로젝트 개요
이 프로젝트는 AST 기반 AI 라우터 시스템으로, Python 코드의 함수 호출 관계를 분석하여 그래프를 구성하고 지능적인 라우팅을 목표로 합니다.

2026-04-07 (그래프 정교화):

그래프 cleaning 로직 추가
스코어링 시스템 구현

CLI + 파이프라인 리팩토링

핵심 아키텍처
1. 데이터 흐름
Python 소스 → AST 분석 → 호출 그래프 → cleaning → 스코어링 → 라우팅
2. 주요 모듈
core/graph_v5.py: AST 기반 함수 호출 그래프 생성 core/graph_clean.py: 그래프 노이즈 제거
core/router_rank.py: 노드 스코어링 및 정렬 core/score.py: 노드 중요도 계산 core/pipeline.py: 실행 파이프라인

3. 현재 구현 단계
✅ Level 1: AST extraction
✅ Level 2: graph construction
✅ Level 3: noisy but usable graph
❌ Level 4: semantic dependency graph
❌ Level 5: AI router graph
4. API 엔드포인트
/api/run: 사용자 입력 → 그래프 검색 → AI 응답 생성

주요 특징
FastAPI 기반 웹 서버
AST를 이용한 정적 코드 분석
그래프 기반 라우팅 시도
실험적 프로젝트로 계속 변경 중
이 시스템은 단순한 코드 분석기를 넘어, 의미 기반의 지능적인 코드 실행 라우터를 목표로 하는 실험적 프로젝트입니다.
