# AI Router Project (ai_router)

## 개요
이 프로젝트는 DAG 기반 노드 실행 엔진 구조를 가진 간단한 AI 라우팅 시스템이다.

노드 간 의존성을 그래프로 정의하고, Topological Sort를 통해 실행 순서를 결정한 뒤 순차적으로 노드를 실행하는 구조다.

---

## 현재 구현 상태 (v0.1)

### 1. Topological Sort 엔진 구현 완료
- Kahn Algorithm 기반 DAG 정렬
- dependency 기반 indegree 계산
- cycle detection 포함

핵심 파일:
- core/toposort.py

---

### 2. Pipeline 구조 (기본 실행 엔진)
- 노드 실행 함수 존재
- dependency 기반 입력 데이터 생성
- 단순 문자열 처리 기반 mock execution

핵심 파일:
- core/pipeline.py

---
